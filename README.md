# BellTower Strategy Group

Modern homepage mockup for BellTower Strategy Group — an independent advisory firm helping employers select, evaluate, and optimize insurance brokers.

## Preview

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Stack

- Semantic HTML
- Custom CSS (Cormorant Garamond + Sora)
- Light vanilla JS for mobile nav and scroll reveals
